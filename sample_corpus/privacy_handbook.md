# Privacy Operations Handbook

## Data Request Response Times

All privacy data requests must be logged on the day they are received. The operations team should acknowledge each request within one business day and complete the first review within five business days. If the request is incomplete, the reviewer must record the missing information and send a clarification request.

## Remote Work And Confidentiality

Staff working remotely must access confidential information only through an approved secure connection. Client files may not be copied to personal devices. Printed records must be locked when not actively used, and disposal must follow the confidential shredding procedure.

## Adding New Corpus Files

New files should be placed into the approved corpus folder. The indexing tool records a file hash for each document, so unchanged files are skipped during later runs. If a source document changes, the new hash causes it to be re-indexed.
